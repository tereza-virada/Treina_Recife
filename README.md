# flzfelipe

Aqui vão 5 comandos básicos do Git que você precisa conhecer:

1. **`git init`**
   Inicializa um novo repositório Git na pasta atual.

2. **`git clone`**
   Faz uma cópia de um repositório remoto para sua máquina.

3. **`git add`**
   Adiciona arquivos para a área de staging (prepara para o commit).

4. **`git commit`**
   Salva as mudanças no histórico do repositório.

5. **`git push`**
   Envia seus commits para um repositório remoto (como o GitHub).
